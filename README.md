# SM-TPS-WPF-MVVM

Store Management & Transaction Processing System built with C#, WPF and MVVM Architecture.

## Features
- Product Management
- Stock Tracking
- Add / Update / Delete Products
- Shopping Cart
- Checkout System
- VAT Calculation
- Transaction History
- Login Screen

## Technologies
- C#
- WPF
- MVVM
- ObservableCollection
- ICommand
