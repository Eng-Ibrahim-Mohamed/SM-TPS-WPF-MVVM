# 🛒 SM-TPS-WPF-MVVM

A modern **Store Management & Transaction Processing System (POS)** built using **C#, WPF, MVVM Architecture, and SQLite Database**.

Designed to simulate a real-world retail environment with inventory management, barcode support, sales tracking, user management, and invoice generation.

---

## ✨ Features

### 🔐 Authentication & Security

* Login System
* Database-Based User Authentication
* User Management
* Change Password Dialog
* Role Support (Admin / Cashier)

### 📦 Product Management

* Add Products
* Update Products
* Delete Products
* Stock Tracking
* Low Stock Alerts
* Barcode Support

### 🛒 Sales & Checkout

* Shopping Cart System
* Quantity Management
* VAT Calculation
* Checkout Processing
* Sales History Tracking
* Transaction Management

### 🧾 Invoice System

* PDF Invoice Generation
* Invoice Viewer
* Invoice Printing

### 📊 Dashboard & Monitoring

* Product Dashboard
* Inventory Monitoring
* Sales Tracking
* Transaction History

---

## 🛠 Technologies Used

* C#
* WPF
* MVVM Architecture
* SQLite Database
* Entity Framework Concepts
* ICommand
* ObservableCollection
* iTextSharp (PDF Generation)

---

## 🏗 Architecture

This project follows the **MVVM (Model-View-ViewModel)** design pattern:

* 📁 Models
* 🖥 Views
* ⚙ ViewModels
* 🗄 Data Layer
* 🔄 Commands
* 📄 Services

This architecture ensures maintainability, scalability, and clean separation of concerns.
